# 导读与学习路线

> 预计阅读：10 分钟 | 前置知识：无

---

## 这个项目是什么

本项目是一个**无人机领域的 VLA/VLM/世界模型学习知识库**，目标是帮助你：

1. **理解三大核心概念**：世界模型（World Model）、视觉语言模型（VLM）、视觉语言动作模型（VLA）
2. **掌握领域全景**：知道这个领域在研究什么、怎么分类、关键论文有哪些
3. **深入无人机方向**：理解这些技术如何应用于无人机，与地面机器人有何不同
4. **获得动手能力**：知道如何复现关键项目，即使暂时不跑代码也能理解流程

---

## 你需要什么前置知识

| 知识领域 | 最低要求 | 理想水平 |
|----------|---------|---------|
| **深度学习** | 知道什么是神经网络、CNN、Transformer | 了解预训练、微调、注意力机制 |
| **计算机视觉** | 知道什么是图像分类、目标检测 | 了解语义分割、光流、视觉里程计 |
| **强化学习** | 知道什么是状态、动作、奖励 | 了解策略梯度、Actor-Critic |
| **自然语言处理** | 知道什么是Token、语言模型 | 了解BERT/GPT架构 |
| **无人机基础** | 知道什么是四旋翼、PID控制 | 了解MAVLink、ROS2、PX4/ArduPilot |

**不需要**：CUDA编程、机器人学深入知识、硬件经验

## 学习路线图

### 阶段一：建立认知框架（1周）

```
目标：理解三大概念是什么、有什么关系
路径：00-导读 → 01-基础概念(5篇)
输出：能用自己的话解释 VLM、VLA、世界模型
```

**推荐阅读顺序**：
1. 什么是世界模型 — 理解"预测未来"的核心思想
2. 什么是VLM — 理解"看图说话"的基础
3. 什么是VLA — 理解从"理解"到"行动"的演进
4. 三者关系与区别 — 串联三者的关系
5. 无人机vs地面机器人 — 理解无人机的特殊性

### 阶段二：选择方向深入（2-3周）

```
目标：深入理解一个方向的论文和方法
路径：02/03/04 专题（选一个或多个）
输出：能分析一篇论文的创新点和局限性
```

**方向选择建议**：

| 方向 | 适合人群 | 核心问题 |
|------|---------|---------|
| **世界模型** | 对预测/仿真/数据生成感兴趣 | 如何让无人机"想象"未来会发生什么 |
| **VLA** | 对端到端控制感兴趣 | 如何让无人机"看+说+飞"一体化 |
| **VLM** | 对感知/理解感兴趣 | 如何让无人机"理解"看到的内容 |

### 阶段三：综述精读（2周）

```
目标：系统理解领域全貌
路径：05-综述论文精读(5篇)
输出：能画出领域的知识图谱
```

### 阶段四：论文导读（2周）

```
目标：掌握30+篇关键论文
路径：06-论文导读合集(4篇)
输出：能快速判断一篇论文的价值和位置
```

### 阶段五：动手实践（持续）

```
目标：能够复现关键项目
路径：07-实践指南(7篇)
输出：能跑通一个项目并理解其架构
```

每个文档遵循统一格式：

```markdown

> 预计阅读：X 分钟 | 前置知识：xxx

---

（配图、配表、配公式简化版）
```

**阅读建议**：
- 不要试图一次读完所有文档，按路线图分阶段进行
- 每篇文档的思考题先自己想，再看答案
- 遇到不懂的术语，先查 [基础概念](01-基础概念/) 部分

## 常见问题

**Q: 我需要跑代码吗？**
A: 不需要。本项目是认知导向的学习项目，重点在理解原理。实践指南（Part 7）告诉你"怎么跑"，但不要求你跑。

**Q: 我应该按什么顺序读？**
A: 先读导读（本文档），再读基础概念（Part 1），然后选择感兴趣的方向深入。详见上方学习路线图。

**Q: 这个项目和原综述论文（arXiv:2605.00080）是什么关系？**
A: 本项目以该综述为骨架，但聚焦于无人机领域，补充了大量无人机专属论文和实践指南。综述面向通用机器人，本项目面向无人机。

**Q: 论文链接打不开怎么办？**
A: 大部分论文在 arXiv 上可以直接访问。如果打不开，可以尝试用论文标题在 Google Scholar 搜索。
