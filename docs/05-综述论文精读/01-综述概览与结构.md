## 综述的核心问题

这篇综述要回答的核心问题是：

> **世界模型如何与机器人学习耦合，推动机器人智能的发展？**

具体包括：
1. 世界模型在机器人学习中扮演什么角色？
2. 世界模型有哪些技术范式？
3. 世界模型如何用于策略学习、仿真评估、视频生成？
4. 有哪些数据集、基准和评估协议？

## 三大角色：世界模型在机器人中的定位

综述将世界模型在机器人学习中的作用归纳为三大角色：

### 角色一：作为策略 (Policy)

世界模型直接充当或辅助机器人策略：

```
传统策略: 观测 → 动作
世界模型策略: 观测 → 未来预测 → 动作
```

**子类别**：
- **IDM风格**：先预测未来视觉轨迹，再用逆动力学模型提取动作
- **单骨干**：用一个统一模型同时建模视频和动作
- **MoE/MoT**：视频和动作专家分离，通过注意力机制交互
- **统一VLA**：将世界建模作为训练目标内化到VLA中
- **隐空间**：在表示空间中做未来预测，不显式生成视频

### 角色二：作为模拟器 (Simulator)

世界模型作为学习环境用于策略改进：

```
学习世界模型 → 在世界模型中"想象"交互 → 改进策略 → 部署到真实环境
```

**两种用途**：
- **强化学习**：Dreamer系列，用世界模型生成训练数据
- **评估**：用世界模型评分候选行为、排序策略、支持MPC规划

### 角色三：作为视频生成器 (Video Generator)

世界模型生成机器人视频用于多种目的：

```
视频生成 → 数据增强 / 想象训练 / 演示生成
```

## 技术分类学

综述沿多个维度对方法进行分类：

| 维度 | 分类 |
|------|------|
| **角色** | 策略骨干 vs 模拟器 vs 视频生成器 |
| **架构** | IDM vs 单骨干 vs MoE/MoT vs 统一VLA |
| **表示** | 像素空间 vs 隐空间 (JEPA风格) |
| **用途** | 训练信号 vs 评估/规划 vs RL微调 |
| **生成方式** | 扩散模型 vs 自回归 vs 流匹配 |

## 关键发现

### 1. 世界模型与策略的耦合越来越紧密

从早期的"世界模型辅助策略"到现在的"世界模型即策略"，边界在模糊。

### 2. 视频生成世界模型快速发展

从想象式生成到可控、结构化、基础规模的公式化。

### 3. 隐空间世界建模兴起

JEPA风格的方法在表示空间中做未来预测，避免了像素空间生成的高成本。

### 4. 基准和评估标准化

综述总结了代表性的数据集、基准和评估协议。

| 综述分类 | 无人机对应 |

| IDM策略 | ANWM的FFP模块 |
| 统一VLA | VLA-AN、CognitiveDrone |
| 世界模型作为模拟器 | Dream to Fly (DreamerV3) |
| 视频生成世界模型 | FlightDiffusion、MotionScape |
| 隐空间世界建模 | 对比世界模型 (无人机导航) |

## 综述的局限性

1. **偏重地面机器人**：大部分例子来自操作和移动，无人机覆盖较少
2. **时间截止**：2026年4月前的工作，可能遗漏最新进展
3. **缺乏实验对比**：作为综述，没有统一的实验评估
4. **代码可用性**：大部分被引用的论文没有开源代码

## 思考题

1. **框架理解**：用自己的话解释世界模型在机器人学习中的三大角色。

2. **分类思考**：IDM策略和单骨干策略的核心区别是什么？各有什么优劣？

3. **无人机适配**：综述的分类框架如何映射到无人机领域？有哪些无人机特有的类别？

4. **趋势判断**：你认为世界模型作为策略 vs 作为模拟器，哪个方向更有前景？

5. **进阶思考**：为什么隐空间世界建模（如JEPA）可能比像素空间生成更适合无人机？
