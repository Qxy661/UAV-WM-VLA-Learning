# 可复现项目候选清单

> 本文档收录与无人机+VLA/VLM/世界模型方向贴合的、可单机复现的开源项目。
> 每个项目标注了难度、硬件需求、预计耗时和与无人机方向的贴合度。
> 供后续选择实际复现时参考。

---

## 选择标准

| 维度 | 说明 |
|------|------|
| **方向贴合** | 必须与无人机/UAV + VLA/VLM/世界模型直接相关 |
| **可复现性** | 有完整代码、文档、数据集，社区活跃 |
| **硬件门槛** | 单 GPU（RTX 4060/4070 级别）或纯 CPU 可跑 |
| **学习价值** | 能帮助理解核心概念，不只是跑通 demo |
| **耗时合理** | 1-7 天可完成，不是需要数周训练的大项目 |

---

## A 类：直接无人机项目（强烈推荐）

### A1. gym-pybullet-drones — 无人机 RL 控制仿真

| 项目 | 信息 |
|------|------|
| **GitHub** | [utiasDSL/gym-pybullet-drones](https://github.com/utiasDSL/gym-pybullet-drones) |
| **Stars** | ~2,000 |
| **硬件需求** | 纯 CPU 即可（PyBullet 物理引擎） |
| **难度** | ★★☆☆☆ 入门级 |
| **预计耗时** | 1-2 天 |
| **贴合度** | ★★★★★ 无人机控制核心 |

**项目简介**：基于 PyBullet 的四旋翼 Gymnasium 仿真环境，支持单机/多机 RL 训练。提供 PID 控制示例和 PPO 训练脚本，可对接 Betaflight/Crazyflie 固件做 SITL 测试。

**为什么推荐**：
- 最贴近无人机方向的 RL 入门项目
- 纯 CPU 运行，零 GPU 门槛
- 代码简洁，文档完善
- 可在此基础上扩展：加世界模型、加 VLA 控制器

**复现内容**：
1. 运行 PID 悬停 demo
2. 用 PPO 训练悬停策略
3. 可选：多机编队、加入视觉观测

---

### A2. safe-control-gym — 安全约束无人机控制

| 项目 | 信息 |
|------|------|
| **GitHub** | [utiasDSL/safe-control-gym](https://github.com/utiasDSL/safe-control-gym) |
| **Stars** | ~879 |
| **硬件需求** | 纯 CPU（测试环境为 i7 无 GPU） |
| **难度** | ★★★☆☆ 中级 |
| **预计耗时** | 2-3 天 |
| **贴合度** | ★★★★★ 安全飞行核心 |

**项目简介**：PyBullet 四旋翼环境 + CasADi 符号动力学，内置 PID/LQR/iLQR/MPC/SAC/PPO 等多种控制器和安全滤波器（MPSC、CBF）。

**为什么推荐**：
- 无人机安全控制是核心研究方向
- 对比多种控制器（经典 vs RL vs MPC）
- 安全约束建模（CBF）是前沿方向
- CPU 可跑，适合理解控制理论

**复现内容**：
1. 对比 PID vs LQR vs MPC 控制效果
2. 训练 SAC/PPO 策略
3. 加入安全约束（CBF）观察效果差异

**项目简介**：北航 CoLab 的语言条件无人机控制项目。提供 UAV-Flow 数据集（真实+仿真轨迹）、OpenVLA-UAV 模型（基于 OpenVLA 微调）、评估环境（基于 UnrealZoo Gym）。

**为什么推荐**：
- 直接对标我们文档中的 VLA 论文
- 有预训练模型可直接推理
- 数据集 + 模型 + 评估环境完整
- 理解 VLA 如何应用于无人机

**复现内容**：
1. 下载数据集，理解数据格式
2. 用 OpenVLA-UAV 做推理 demo
3. 可选：在仿真环境中评估

### A4. GeoChat — 遥感 VLM 对话

| 项目 | 信息 |
|------|------|
| **GitHub** | [mbzuai-oryx/GeoChat](https://github.com/mbzuai-oryx/GeoChat) |
| **Stars** | ~713 |
| **硬件需求** | 推理需 GPU（~8GB），LoRA 微调需 ~16GB |
| **难度** | ★★★☆☆ 中级 |
| **预计耗时** | 2-3 天 |
| **贴合度** | ★★★★☆ 无人机遥感理解 |

**项目简介**：CVPR 2024 首个遥感 VLM，基于 LLaVA-1.5 + LLaMA-2，支持遥感图像对话、VQA、目标检测、场景分类。有 LoRA 微调代码。

**为什么推荐**：
- 遥感 VLM 直接适用于无人机视角
- 有预训练模型，可直接推理
- LoRA 微调流程完整，学习 VLM 训练
- CVPR 2024 论文，质量有保证

**复现内容**：
1. 用预训练模型做遥感图像对话
2. 在自己的无人机图片上测试
3. 可选：LoRA 微调适配特定场景

## B 类：通用但可迁移到无人机（推荐）

### B1. LeRobot — 机器人学习框架

| 项目 | 信息 |
|------|------|
| **GitHub** | [huggingface/lerobot](https://github.com/huggingface/lerobot) |
| **Stars** | ~23,900 |
| **硬件需求** | 训练需 GPU（~8GB），推理可 CPU |
| **难度** | ★★★☆☆ 中级 |
| **预计耗时** | 2-4 天 |
| **贴合度** | ★★★☆☆ 通用机器人→可迁移到无人机 |

**项目简介**：HuggingFace 机器人学习框架，支持 ACT、Diffusion Policy、Pi0Fast、SmolVLA 等多种策略。有标准化数据集格式和仿真评估（LIBERO、MetaWorld）。

**为什么推荐**：
- 代码质量极高，HuggingFace 维护
- 支持多种 VLA 架构（Pi0Fast、SmolVLA）
- 学习端到端机器人策略训练流程
- 可扩展到无人机场景

**复现内容**：
1. 跑通 ACT/Diffusion Policy 在 LIBERO 上的 demo
2. 理解 LeRobotDataset 数据格式
3. 思考如何迁移到无人机控制

|------|------|
| **GitHub** | [ultralytics/ultralytics](https://github.com/ultralytics/ultralytics) |

| **贴合度** | ★★★☆☆ 无人机视觉感知基础 |

**项目简介**：YOLO 系列最新版本，支持旋转目标检测（OBB），已在 DOTA 航拍数据集上预训练。

**复现内容**：
1. 用预训练 YOLO-OBB 在 DOTA 上做推理
2. 在自己的航拍图片上测试
3. 可选：微调适配特定无人机场景

### B3. OpenVLA 微调 — VLA 模型理解

| 项目 | 信息 |
|------|------|
| **GitHub** | [openvla/openvla](https://github.com/openvla/openvla) |
| **Stars** | ~6,100 |
| **硬件需求** | LoRA 微调需 ~24GB（A100/4090），推理 ~16GB |
| **难度** | ★★★★☆ 高级 |
| **预计耗时** | 3-5 天 |
| **贴合度** | ★★★★☆ VLA 核心架构理解 |

**项目简介**：UC Berkeley 开源 VLA 模型（7B 参数），基于 Prismatic VLM + 动作 token 化。支持 LoRA 微调和 REST API 部署。

**为什么推荐**：
- 理解 VLA 架构的最佳实践
- LoRA 微调流程完整
- UAV-Flow 的 OpenVLA-UAV 就是基于此微调
- 为后续无人机 VLA 微调打基础

**复现内容**：
1. 下载预训练模型，跑通推理 demo
2. 理解 VLA 的输入输出格式
3. 可选：LoRA 微调（需较大 GPU）

### B4. CogVLM — 通用 VLM 用于航拍理解

| 项目 | 信息 |
|------|------|
| **GitHub** | [THUDM/CogVLM](https://github.com/THUDM/CogVLM) |
| **Stars** | ~6,700 |
| **硬件需求** | 推理需 ~16GB GPU |
| **难度** | ★★★☆☆ 中级 |
| **预计耗时** | 1-2 天 |
| **贴合度** | ★★★☆☆ 通用 VLM→可测航拍 |

**项目简介**：清华开源 VLM（10B 视觉参数 + 7B 语言参数），支持图像描述、VQA、视觉定位。

**为什么推荐**：
- 开源 VLM 中质量较高
- 可直接测试无人机航拍图片理解
- 有微调 demo，可学习 VLM 训练
- 对比 GeoChat 看通用 vs 领域专用差异

**复现内容**：
1. 用预模型对航拍图片做描述/VQA
2. 对比通用 VLM vs 遥感 VLM（GeoChat）效果
3. 可选：在无人机数据上微调

## 推荐复现路径

```
第1天: A1 gym-pybullet-drones（PID + PPO 悬停）
         ↓ 理解无人机 RL 基础
第2天: A2 safe-control-gym（对比控制器 + 安全约束）
         ↓ 理解安全控制
第3天: B2 YOLO-OBB 航拍检测
         ↓ 理解无人机视觉感知
第4-5天: A4 GeoChat 遥感 VLM 推理
         ↓ 理解 VLM 在遥感中的应用
第6-7天: A3 UAV-Flow VLA 推理
         ↓ 理解 VLA 无人机控制
可选进阶: B1 LeRobot 或 C1 Dreamer+无人机
```
